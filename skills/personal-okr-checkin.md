# OKR Monthly Check-in

## Location

OKR file: `~/Documents/Personal/20-29 Personal/24. Goals and improvement/24.05 Yearly theme and goals/2026 Theme, objectives, and key results.md`

## Process

1. Read OKR file to get all key results and Q1 goals
2. For each KR, use AskUserQuestion to gather progress
   - Check if data exists in Obsidian first (blog posts, etc.) before asking
   - One question at a time
3. Summarize results in a table (KR | Progress | Status)
4. Fill in the monthly review section using the template structure:
   - Add "Action items for [next month]" section at top with empty bullets
   - For each KR: Q1 Goal, Progress, Status (just the status text, no checkboxes)
   - Add Notes field only when off track

## Status values

- **On track**: Meeting or exceeding pace
- **Off track**: Behind pace with limited recovery time
- **Not started**: Q1 goal not yet begun (still recoverable)
